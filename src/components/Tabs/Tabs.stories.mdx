## Usage

### Default tabs

Use for most cases, especially when the number of tabs may be more than three.

### Static tabs

Use when it's easier to define the tabs via JSX. Each tab item will be wrapped in a Tab
component and is responsible for providing an `onClick` handler that updates the parent
`selected` prop.
