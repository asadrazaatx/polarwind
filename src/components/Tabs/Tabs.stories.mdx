## Usage

### Default tabs

Use for most cases, especially when the number of tabs may be more than three.

### Static tabs

Use when it's easier to define the tabs via JSX. Each tab item will be wrapped in a Tab
component and is responsible for providing an `onClick` handler that updates the parent
`selected` prop.

<Preview withToolbar>
  <Story name="Static tabs">
    {() => {
      const [selected, setSelected] = useState(0);
      return (
        <Tabs selected={selected} onSelect={setSelected}>
          <span>Single invite</span>
          <span>Group invite</span>
          <a href="https://envoy.com">Go to Envoy</a>
        </Tabs>
      );
    }}
