## Usage

### Default

AppProvider works by default without any additional options passed to it.

### With theme

With a `theme`, the app provider component will set theming for the App. For theming
configuration, see the `ThemeProvider` documentation.

<Preview withToolbar>
  <Story name="With theme">

    </AppProvider>
