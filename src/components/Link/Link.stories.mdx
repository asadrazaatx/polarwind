## Usage

### Default link

Use for text links in larger spans of text.

### Underlined link

Use when set against a background that is similar in color to give it visual contrast.

<Preview>
  <Story name="Underlined">
    <Link url="https://dashboard.envoy.com/deliveries" underlined>
      Give it a try â†’
    </Link>
  </Story>
</Preview>

### Monochrome link

Use for text links that are the same color as the surrounding text.

<Preview>
  <Story name="Monochrome">
    <div>
      <p>
        Monochrome links get underlines so that we you can differentiate{" "}
        <Link url="https://dashboard.envoy.com/deliveries" monochrome>
          what's a link
        </Link>{" "}
        and what's just text.
      </p>
      <Link
        url="https://dashboard.envoy.com/deliveries"
        monochrome
        underlined={false}
      >
        You can also turn off underlines if you really wanted
      </Link>
    </div>
  </Story>
</Preview>

### External link

Use for text links that should open in a new browser tab (or window, depending on the
user's browser settings).

When linking to an external URL while embedded, this prop is required or the browser will
refuse to follow the link if the site has `X-Frame-Options` set to disallow being rendered
in an iframe.

<Preview>
  <Story name="External">
    <Link url="https://www.google.com" external>
      Google
    </Link>
  </Story>
</Preview>

### Link without a URL

Use when presenting a text link that performs an action via the `onClick` callback.

<Preview>
  <Story name="No URL">
    <Link onClick={() => alert("hi")}>Alert</Link>
  </Story>
</Preview>
