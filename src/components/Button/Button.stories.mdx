## Usage

### Sizes

Apart from the standard button, we support slim and large variant buttons.

<Preview>
  <Story name="Sizes">
    <div className="flex justify-around">
      <Button size="slim">Add product</Button>
      <Button size="large">Add product</Button>
    </div>
  </Story>
</Preview>

### Styles

An outline button will maintain the appropriate visual weight and won't clash with the
background color. For secondary actions, the outline style is appropriate while the plain
style can be used for tertiary actions.

<Preview>
  <Story name="Styles">
    <div className="flex justify-around">
      <div>
        <ButtonGroup>
          <Button>Invite</Button>
          <Button outlineBrand>Invite</Button>
          <Button outline>Invite and add another</Button>
        </ButtonGroup>
      </div>
      <Button plain>Cancel</Button>
    </div>
  </Story>
</Preview>

### Disabled

Used to indicate that the button is not clickable. This is used in situations where a
condition has not been fulfilled, for example, when a form has a validation error.

<Preview>
  <Story name="Disabled">
    <div className="flex justify-around">
      <Button disabled>Add product</Button>
      <Button outlineBrand disabled>
        Add product
      </Button>
      <Button outline disabled>
        Add product
      </Button>
      <Button plain disabled>
        Add product
      </Button>
    </div>
  </Story>
</Preview>

### Icon

`Button` accepts an `icon` prop to display an icon on the left of the button. It is only
supported on the slim and normal buttons.

<Preview>
  <Story name="Icon">
    <div className="flex justify-around">
      <Button size="slim" icon={FiTrash2}>
        Delete
      </Button>
      <Button icon={FiTrash2}>Delete</Button>
      <Button size="large" icon={FiTrash2}>
        Delete
      </Button>
    </div>
  </Story>
</Preview>

### URL

`Button` accepts a `url` prop when presenting an actionable link (as oppossed to a
navigational link)

<Preview>
  <Story name="URL">
    <div className="flex justify-around">
      <Button url="https://www.google.com">Visit</Button>
      <Button disabled url="https://www.google.com">
        Visit
      </Button>
      <Button outlineBrand url="https://www.google.com">
        Visit
      </Button>
      <Button outlineBrand disabled url="https://www.google.com">
        Visit
      </Button>
      <Button url="https://www.google.com" outline>
        Visit
      </Button>
      <Button disabled url="https://www.google.com" outline>
        Visit
      </Button>
      <Button url="https://www.google.com" plain>
        Visit
      </Button>
      <Button disabled url="https://www.google.com" plain>
        Visit
      </Button>
    </div>
  </Story>
</Preview>
